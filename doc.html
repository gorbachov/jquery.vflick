Setting Options

form of hash in the argument of vflick method.

Options

key default type
verticalFlick false boolean horizontal or vertical flick setting
flickClass .flick string class name of each flick elements.
transitionSpeed 350 number transition speed of flick animation.
transitionTiming cubic-bezier(0, 0, 0.25, 1) string transition timing of flick animation.
ex) ease-out, ease-in, etc..
fsEndAnim function(){} function callback function of finish at flick animation.
current page number is returned to the argument.

methods

goNext

method to advance the next page

goPrev

method to return the prev page
